Click to select.

Alt+C - Add circle.

Alt+A - Add path.

Alt+R - Add rect.

Alt+Shift+D - Delete selection.

V - Add point/vertex to path.

Shift - Enable snapping (defaults to grid).

Alt+' - Toggle snap to path points.

Alt+click - Remove point from path.

Alt+Space - List canvas calls in console.

Alt+Shift+Space - Display JSON.

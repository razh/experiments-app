Click to select.

Arrow keys - Move selection (shift to move by grid units).

Alt+A - Add arc.

Alt+C - Add circle.

Alt+E - Add path.

Alt+R - Add rect.

Alt+D - Duplicate selection.

Alt+P - Toggle path drawing mode.

Backspace/Delete - Delete selection.

V - Add point/vertex to path.

Shift - Enable snapping (defaults to grid).

Alt+S - Toggle snap to path points.

Alt+Shift+Click - Remove point from path.

Alt+Shift+C - Center path on centroid.

Alt+Space - List canvas calls in console.

Alt+Shift+Space - Display JSON.
